# 라스트티 (LastTee)

**취소된 골프 자리, 반값 참여.**
일행 취소로 비어버린 티타임을 실시간으로 올리고, 할인가로 참여하는 라스트미닛 골프 플랫폼입니다.

## 왜 라스트티인가

한국 골프는 비쌉니다. 그린피에 캐디피, 카트비까지 드는데 골프장은 4인 예약이 원칙이라 한 명만 빠져도 남은 일행이 위약금과 미달 그린피를 떠안습니다.

* **호스트**: 위약금을 무느니 빈자리를 할인가로 양도해 손해를 줄입니다.
* **참여자**: 혼자서는 못 가던 라운드를 반값에 즐깁니다.
* **골프장**: 노쇼와 공티 손실이 매출로 바뀝니다. 제휴(B2B) 확장 여지가 있습니다.

## 주요 기능

* **빈자리 지도**: 실제 대한민국 행정경계 지형 위에 실시간 모집 핀 표시
* **실제 코스 전경**: 골프장마다 실제 위성 이미지(Esri World Imagery)로 코스 확인
* **네이버 지도 연동**: 골프장 상세에서 바로 네이버 지도로 이동
* **마감 임박 카운트다운**: 티오프까지 남은 시간 실시간 표시, 1시간 이내 긴급 표시
* **투명한 1/N 정산**: 그린피, 캐디피, 카트비 분담 내역과 절약액 표시
* **결제 수단**: 현장결제 또는 계좌이체 선택, 호스트 정산 계좌 등록과 자동 안내
* **다이렉트 메시지**: 호스트와 1:1 채팅, 참여 전 문의와 정산 안내
* **프로필과 그린지수**: 본인인증, 라운드 히스토리, 동반자 상호 후기 기반 신뢰 점수
* **노쇼 방지**: 24시간 전 무료취소, 이후 위약금, 노쇼 3회 이용 제한
* **골프 크루**: 정기 라운드 커뮤니티, 피드, 좋아요
* **설정**: 알림 설정, 다크 모드, 쿠폰함, FAQ, 공지, 약관, 친구 초대, 앱 설치 가이드
* **PWA**: iOS와 Android 홈 화면 설치, 리퀴드 글래스 플로팅 탭바, 오프라인 셸 캐시

## 스택

바닐라 HTML/CSS/JS 단일 페이지 앱. 빌드 없음. GitHub Pages 배포.
데모 데이터는 localStorage 기반이며 백엔드는 추후 연동 예정입니다.
지도 데이터는 `make_geo.mjs`가 실제 행정경계 GeoJSON과 골프장 좌표로 생성합니다.

## 로드맵

* 실 골프장 제휴 API (잔여 티타임 직판)
* 앱 내 결제와 에스크로 (노쇼 보증금)
* 실시간 채팅과 푸시 알림
* 카카오 로그인과 실 본인인증 (PASS)
* 크루 개설과 크루 전용 단체 할인

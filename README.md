# ⛳ 라스트티 (LastTee)

**취소된 골프 자리, 반값 조인.**
일행 취소로 비어버린 티타임을 실시간으로 올리고, 할인가로 조인하는 라스트미닛 골프 조인 플랫폼.

## 왜 라스트티인가

한국 골프는 비쌉니다. 그린피 + 캐디피 + 카트비에, 골프장은 4인 예약이 원칙이라 **한 명만 빠져도 남은 일행이 위약금·미달 그린피를 떠안습니다.**

- **호스트**: 위약금을 무느니 빈자리를 할인가로 양도 → 손해 최소화
- **조이너**: 혼자서는 못 가던 라운드를 반값에 → 진입장벽 해소
- **골프장**: 노쇼·공티 손실이 매출로 전환 → 제휴(B2B) 확장 여지

## 주요 기능

- 🗺️ **빈자리 지도** — 전국 골프장 위 실시간 조인 현황 (도트맵 + 핀)
- ⏱️ **마감 임박 카운트다운** — 티오프까지 남은 시간 실시간 표시
- 💸 **투명한 1/N 정산** — 그린피·캐디피·카트비 분담 내역과 절약액 표시
- ⚡ **즉시확정 / 승인제** 조인, 참여 조건(평균 타수)·성별 우대 필터
- 👤 **프로필 & 그린지수** — 본인인증, 라운드 히스토리, 동반자 상호 후기(별점) 기반 신뢰 점수
- 🚫 **노쇼 방지** — 24시간 전 무료취소, 이후 위약금, 노쇼 3회 이용 제한
- 🤝 **골프 크루** — 조인으로 만난 사람들이 커뮤니티로: 정기 라운드, 피드, 번개
- 🏌️ **골프장 상세** — 홀·파·전장·요금표·시설·코스 뷰
- 📣 **모집 올리기** — 30초 만에 빈자리 등록, 할인율 코칭
- 📱 **PWA** — 홈 화면 설치, 오프라인 셸 캐시

## 스택

바닐라 HTML/CSS/JS 단일 페이지 앱. 빌드 없음. GitHub Pages 배포.
데모 데이터는 localStorage 기반 (백엔드 추후 연동).
